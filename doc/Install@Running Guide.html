Instructions for Installing and Running the Project

1. Clone the Repository:

git clone https://github.com/yourusername/currency_converter_api.git

2. Navigate to the Project Directory:

cd currency_converter_api

3. Build and Run the Project:

docker-compose -f docker-compose.yml up --build -d

4. Apply Migrations:

docker-compose -f docker-compose.yml exec app alembic upgrade head

This step ensures that the database is initialized with the necessary tables and data.

Adjust these commands based on your actual project structure and requirements.
